# Lumina
## a ChatGPT-like system to interface with your LM Studio backend

This is a **beta project**. Because of that it will probably be very buggy.

This was fully made using Kimi K2.5 Thinking via Moonshot AI's free usage tier (https://www.kimi.com/).

# Features
- Multiple chats, saved to localStorage
- Memories injected a-la-ChatGPT into your model; can be saved by it & edited by you
- Thinking (if available) can be selected
- Can add personal details (name, pronouns, etc.)
- API Key support (you can add your LM Studio API key)
